# 競標區功能說明

## 功能概述

競標區是群內成員專屬的限時競標活動，讓成員可以競標稀有的配布寶可夢。

## 用戶角色

| 角色 | 權限 |
|------|------|
| `user` | 一般用戶，可瀏覽但不能出價 |
| `member` | 群內成員，可瀏覽和出價 |
| `admin` | 管理員，可建立/管理競標 |

---

## 競標規則

### 出價機制
- **起標價**：管理員設定的最低起標金額
- **最低加價**：每次出價必須比當前最高價高出的最低金額（預設 +$100）
- **快速加價**：提供 +$100、+$500、+$1000 快速按鈕
- **手動輸入**：可自訂出價金額（需 ≥ 最低出價）

### 結標方式
- **限時結標**：競標於設定的結束時間自動結標
- **最高出價者得標**：結標時出價最高者獲得商品

---

## 頁面結構

### 前台頁面
| 路徑 | 說明 |
|------|------|
| `/auctions` | 競標列表（進行中 + 已結標） |
| `/auctions/[id]` | 競標詳情 + 出價介面 |
| 首頁 | 熱門競標區塊 |

### 管理後台
| 路徑 | 說明 |
|------|------|
| `/admin/members` | 會員角色管理（升降級 member） |
| `/admin/auctions` | 競標管理（建立/編輯/刪除） |

---

## 資料庫結構

### auctions 表
```sql
- id: UUID
- distribution_id: 關聯配布圖鑑
- title: 競標標題
- starting_price: 起標價
- min_increment: 最低加價
- current_price: 目前最高價
- current_bidder_id: 最高出價者
- end_time: 結束時間
- status: draft / active / ended / cancelled
- bid_count: 出價次數
```

### bids 表
```sql
- id: UUID
- auction_id: 關聯競標
- user_id: 出價者
- amount: 出價金額
- created_at: 出價時間
```

---

## 使用流程

### 管理員
1. 前往 `/admin/members` 升級成員為 `member`
2. 前往 `/admin/auctions` 建立競標
3. 選擇配布寶可夢 → 設定價格/時間 → 發布

### 群內成員
1. 瀏覽 `/auctions` 查看進行中的競標
2. 點擊競標進入詳情頁
3. 使用快速加價或手動輸入出價
4. 等待結標結果
